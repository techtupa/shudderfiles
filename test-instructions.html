Test instructions for Meta reviewers

  1. Open Graph API Explorer: Graph API Explorer.
  2. Select the app: AutoPublishShudder.
  3. Generate a user access token (type: user). Use redirect URI https://techtupa.github.io/shudderfiles/auth-callback.html if performing OAuth.
  4. Test endpoints: /me?fields=id,name, /me/accounts, and /{ig_user_id}/media (documentation supplied on app).
