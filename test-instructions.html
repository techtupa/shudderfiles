Testing Instructions

Guide for reviewers testing Meta or TikTok integrations via Shudder Files.

About This App

Shudder Files is an automation tool developed by TechTupa (MEI) to assist creators in publishing short videos and images to Instagram, Facebook, and TikTok through their official APIs. It uses authorized scopes and secure tokens for content publication and analytics access.

OAuth Login & Redirect

  • OAuth Redirect URI: https://techtupa.github.io/shudderfiles/auth-callback.html
  • Test user accounts are pre-authorized and added to the app configuration in the developer console.
  • Log in using a test Meta Business or TikTok Creator account with publishing permissions.
  • On success, the token is used for posting test media and retrieving basic profile data.

Supported Scopes

Meta (Instagram/Facebook):

  • pages_show_list, pages_read_engagement
  • instagram_basic, instagram_content_publish
  • publish_video, pages_manage_posts

TikTok:

  • user.info.profile, user.info.stats
  • video.list, video.upload
  • artist.certification.read, artist.certification.update

Test Steps

  1. Log in using one of the registered test accounts.
  2. Authorize access when the OAuth window prompts for scopes.
  3. Prepare a sample video or image for posting (e.g., a short clip or static thumbnail).
  4. Use the Shudder Files interface or API endpoint to submit the media.
  5. Verify that the post appears on the selected test profile (Facebook Page, Instagram Professional Account, or TikTok profile).

Troubleshooting

  • Check token expiration and refresh flow.
  • Verify connected Page/Account has proper publishing permissions.
  • Ensure that you're using a test user added in the developer console.
